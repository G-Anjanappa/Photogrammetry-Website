WEB MAPPING PROCESSES

According to wikipedia, Web Mapping is the process of using the maps delivered by geographic information systems (GIS) on the Internet, more specifically in the World Wide Web (WWW). It includes the process of designing, implementing, generating, and delivering maps[1].The figure below shows a basic workflow of web mapping.


Web Page/ Web Browser

The content in our web page is specified with three different languages:

  • HTML (Hypertext Markup Language) specifies the content of the web page.
  • CSS (Cascading Style Sheets) specifies the style and presentation of the content.
  • JavaScript is a programming language that defines how that content interacts with users.

For web mapping, the HTML, CSS and JavaScript code in a web page often bring other image content into the web page and make requests to servers as users interact with those web pages.
